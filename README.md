# LARAWIND PORTFOLIO

Create a portfolio site with the Laravel Framework and Tailwind CSS.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

The things you need before installing the project.

* NodeJS
* Xampp, Wampp and etc.
* Google Chrome, Mozilla Firefox, Brave Browser and etc.

### Installation

A step by step guide that will tell you how to get the development environment up and running.

```
$ cp .env.example .env (edit file .env)
$ php artisan config:cache
$ php artisan key:generate
$ php artisan migrate --seed
$ php artisan serve
$ npm run watch (Run scripts from package.json when files change) -New tab
```
